 # ✈️ Travel Journies

**Travel Journies** is a premium, high-fidelity travel diary web application designed to help explorers capture, organize, and relive their adventures.  

Built with a **dark-mode-first aesthetic**, it combines modern web technologies with a smooth, tactile “postcard-style” user experience.

---

## ✨ Key Features

### 🌑 Dark Mode First
A sleek, high-contrast dark theme by default, with a seamless toggle for light mode users.

### 📸 Multi-Media Logs
Upload multiple images per journey with an interactive mini-carousel to browse memories.

### 📍 Smart Location Mapping
One-click integration with Google Maps to view where each journey took place.

### ❤️ Interactive Favorites
Mark your favorite memories and filter your gallery to show only them.

### 📦 Intelligent Storage
- **Persistence:** Uses `localStorage` — no backend required  
- **Capacity:** Supports up to 20 journeys for optimal performance  

### 🎭 Smart Placeholders
If no images are uploaded, the app automatically assigns a curated travel image.

---

## 🛠️ Tech Stack

- **Structure:** HTML5 (Semantic)
- **Styling:** Tailwind CSS (Utility-first design)
- **Logic:** Vanilla JavaScript (ES6+)
- **Icons:** FontAwesome 6
- **Typography:** Outfit & Caveat
