✈️ Travel Journies

Travel Journies is a premium, high-fidelity travel diary web application designed to help explorers capture, organize, and relive their adventures. Built with a "dark-mode first" aesthetic, it combines modern web technologies with a tactile "postcard" UX.

✨ Key Features

🌑 Dark Mode First: A sophisticated, high-contrast dark theme by default, with a seamless toggle for Light Mode enthusiasts.

📸 Multi-Media Logs: Upload multiple images per journey. Includes an interactive mini-carousel within each card to swipe through your gallery.

📍 Smart Location Mapping: One-click integration with Google Maps to view exactly where your memory was made.

❤️ Interactive Favorites: Pin your most cherished memories with a dedicated heart system and filter your gallery to show only favorites.

📦 Intelligent Storage: - Persistence: Uses localStorage to keep your data safe without needing a backend.

Capacity: Built-in tracker to manage a 20-journey limit (optimized for performance).

🎭 Smart Placeholders: No photo? No problem. The app automatically assigns a curated travel landscape if no images are uploaded.

🛠️ Tech Stack

Structure: HTML5 (Semantic)

Styling: Tailwind CSS (Modern Utility-First Design)

Logic: Vanilla JavaScript (ES6+)

Icons: FontAwesome 6.0

Typography: Outfit (Sans-serif) & Caveat (Handwritten style for locations)

📝 Usage Rules

Note Limit: Max 20 entries to ensure lightning-fast local performance.

Image Size: Optimized for images under 2MB (uses Base64 encoding).

Persistence: Data is saved in your browser's local storage. Clearing your browser cache will remove your journies.
